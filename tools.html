使っているツール・サービス

このサイトの制作・運営や、AIでの作業効率化、就職活動、そして投資学習に実際に使っているツールとサービスをご紹介します。

※本ページはプロモーションを含みます。

何もない0の状態からWebで収益を生み出し、投資の勉強を進めるために、たくさんの無料・有料ツールを組み合わせて利用しています。将来的にアフィリエイトリンクなどの紹介報酬を設定できるよう、あらかじめこちらのページにまとめています。

💻 Web制作・開発ツール

サイトの構築、デプロイ、アクセス解析、キーワード調査などに欠かせない開発・運営ツールです。

💻

VS Code

Microsoft製の無料・軽量な高機能エディタ。HTMLやCSSのコーディング、GitHubと連携したプロジェクト管理で使用しています。

公式サイトを見る
📂

GitHub Pages

GitHubが提供する静的ウェブサイトのホスティングサービス。このサイトを世界中に高速で公開するサーバーとして利用しています。

公式サイトを見る
📊

Google Analytics (GA4)

サイト訪問者の行動を可視化するアクセス解析ツール。どのページがどれくらい読まれているかを分析し、改善に活かしています。

公式サイトを見る
🔍

Google Search Console

Google検索経由での流入キーワードやエラーを調査・監視するツール。読者が何を求めて当サイトに来ているかを知る手がかりです。

公式サイトを見る

🤖 AIツール

自己PRのブラッシュアップ、コードのエラー解消、学習時の壁打ち相手として導入している最先端のAIツールです。

💬

ChatGPT

OpenAIの対話型AI。就職活動の自己分析におけるアイデア出しや、ES（エントリーシート）の言い回し添削、投資勉強の質問回答に活用しています。

公式サイトを見る
🧠

Claude

Anthropicが開発した高い文章作成能力を持つAI。自然で読みやすい日本語の執筆アドバイスや、複雑なロジック整理に向いています。

公式サイトを見る

💼 就活サービス

新卒の就職活動における企業情報の収集や、エントリーシート対策、選考スケジュール管理のために使っている代表的なサービスです。

🏢

新卒就活ナビ (仮称)

多数の企業情報、業界研究コンテンツ、採用セミナー情報を網羅した定番の総合就職情報サイトです。

サービス詳細（準備中）
📷

ES添削カメラ (仮称)

スマートフォンのカメラで撮影したES原稿を送ることで、AIや専門アドバイザーからフィードバックを受けられる添削サポートです。

サービス詳細（準備中）
✉️

オファーボックス風逆求人 (仮称)

自身のプロフィールや自己PRを登録しておくことで、興味を持った企業から直接スカウトメッセージが届く逆求人型サービスです。

サービス詳細（準備中）

📈 投資学習リソース

株式取引やNISAなどの税制優遇制度、市場の仕組みを偏りなく正確に学ぶために参照している公式サイトです。
