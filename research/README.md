# AI投資研究

世界から自由に選ぶ「攻めのAI」5社と「守りのAI」5社を、全世界平均（VT）と8週間比較する公開研究のデータ置き場です。実資金は使いません。

## 第1期の流れ

1. 情報締切を固定し、同じ資料を攻め・守りへ渡す。
2. `prompts/` の固定文を別コンテキストで一度ずつ実行する。
3. 回答全文、モデル名、実行日時を保存する。
4. 選定銘柄と予備候補の参加資格を確認する。
5. `predictions.csv` へ確定した10社とVTを記録する。
6. 開始時の調整後終値と為替を100として8週間追跡する。
7. 毎週土曜00:00 UTC時点の直前終値で `valuations.csv` を更新する。
8. `python research/evaluate.py` で整合性と成績を確認して公開する。

## 変更しないもの

- 各チームの仮想元本は10万円。
- 攻め・守りは各5社へ20%ずつ配分する。
- 選定後8週間は銘柄を変更しない。
- 攻めと守りの重複は、そのままAIの判断結果として扱う。
- 回答を気に入るまで再生成しない。
- 第1期途中のルール変更は第2期へ送る。

## 主なファイル

- `protocol.json`: 固定ルール。第1期開始後は変更しない。
- `prompts/attack.txt`: 攻めのAIへ渡す固定プロンプト。
- `prompts/defense.txt`: 守りのAIへ渡す固定プロンプト。
- `predictions.csv`: 銘柄の正式情報、選定理由、資格確認。
- `valuations.csv`: 週次の円換算評価額と現地通貨成績。
- `evaluate.py`: 入力検証とチーム成績の集計。

取得した価格・為替の生データは取得日時とともに `data/raw/` へ保存します。データ欠損、30%以上の価格変化、古い終値、通貨不明があれば公開処理を止め、公式情報または第二情報源で確認します。

これは特定モデルと固定プロンプトによる一度の探索的な仮想研究です。投資助言、売買推奨、将来の利益を示すものではありません。
