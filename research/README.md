# 0円AI投資研究所

実資金を使わず、AIの判断が単純なルールやTOPIXと比べてどうだったかを8週間記録するためのデータ置き場です。これは投資助言やバックテストによる利益保証ではありません。

## 第1期の固定ルール

- 期間: 8週間
- 仮想元本: 各方式100,000円
- 方式: `ai`、`rule`、`topix`
- 判定期間: 原則として月曜始値から金曜終値（休場時は各営業日の実値）
- 配分: 方式ごとに均等配分
- 空売り: なし
- 見送り: `cash` として記録し、リターン0%
- 往復コスト: 0.2%（ポジションを持った行のみ）
- 途中のルール変更: 禁止。変更は次期研究として扱う

厳密な条件は `protocol.json` を正とします。

## 記録手順

1. 予測時点で `predictions.csv` に週、方式、対象、方向、根拠、入力データの確認時刻を記録する。
2. 同じコミットで公開し、予測を後から書き換えない。
3. 判定後に開始価格・終了価格・根拠確認結果を追記する。
4. `python research/evaluate.py` を実行する。
5. 出力された集計を研究ページへ反映する。

訂正が必要な場合は元の行を消さず、`status=corrected` として訂正行を追加し、Git履歴と理由を残します。

## CSVの主な列

- `direction`: `up`、`cash` のいずれか。第1期は空売りを扱いません。
- `entry_price` / `exit_price`: 分割調整済みの同一基準価格。
- `evidence_check`: `pass`、`fail`、`pending`。AIの根拠を入力データ・一次情報と照合した結果。
- `status`: `planned`、`settled`、`corrected`。
- `notes`: 欠損、休場、訂正理由など。

株価データの取得元・取得日時・生データは `data/raw/` に保存します。取得ライブラリだけに依存せず、参照元URLも記録してください。
