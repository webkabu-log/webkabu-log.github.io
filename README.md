# Web収益だけで株を買う日記

大学生がAIとWebを使い、0円から収益を作って投資を学ぶ過程を公開する静的Webサイトです。投資助言ではなく、個人の学習・運営記録として、成功だけでなく迷い、未達、失敗、収益や損失も記録します。最初の目標は「Webで1円を作る」です。

## 公開先

- 公開サイト: https://webkabu-log.github.io/
- GitHub: https://github.com/webkabu-log/webkabu-log.github.io
- ローカル作業場所: `C:\trader`

## ローカルで確認する

このプロジェクトは静的サイトです。`C:\trader` をカレントディレクトリにしてローカルサーバーを起動し、ブラウザで `http://localhost:8000/` を開きます。

```powershell
cd C:\trader
python -m http.server 8000
```

## 主要ドキュメント

- [PROJECT_RULES.md](PROJECT_RULES.md): プロジェクト全体のルールと制約
- [FAILURE_LOG.md](FAILURE_LOG.md): 失敗、原因、再発防止策の記録
- [NEXT_ACTIONS.md](NEXT_ACTIONS.md): 次に行う作業と優先順位
- [CONTENT_STYLE_GUIDE.md](CONTENT_STYLE_GUIDE.md): 記事の文章、構成、見せ方のルール

作業前に `PROJECT_RULES.md` と `NEXT_ACTIONS.md` を確認し、記事を編集するときは `CONTENT_STYLE_GUIDE.md` も参照してください。
