# Web収益だけで株を買う日記

Webで生まれた収益だけを使って、少額投資に挑戦する個人学習記録サイトです。

- 公開サイト: https://webkabu-log.github.io/
- GitHub: https://github.com/webkabu-log/webkabu-log.github.io
- ローカル作業場所: `C:\trader`

## コンセプト

生活費、貯金、借金は使わず、Webサイトから生まれた収益だけを投資資金にする。
最初の目標は「Webで1円を作る」こと。

このサイトは投資助言ではなく、翔がWeb収益化と少額投資を学んでいく記録です。

## 作業を再開する手順

別端末では、まずリポジトリを取得します。

```powershell
git clone https://github.com/webkabu-log/webkabu-log.github.io.git
cd webkabu-log.github.io
```

変更したら、以下の流れでGitHubへ反映します。

```powershell
git status
git add .
git commit -m "Update site"
git push
```

詳しい引き継ぎ内容は [HANDOFF.md](HANDOFF.md) を見てください。
